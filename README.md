Часть 1: Исследование хеш-функций

Программа анализирует три популярных хеш-алгоритма:
MD5	16 байт (128 бит)	Устаревший, не рекомендуется для безопасности;
SHA-1	20 байт (160 бит)	Ослаблен, не рекомендуется для новых систем;
SHA-256	32 байта (256 бит)	Современный стандарт, криптостойкий.

Что проверяется:
✅ Хеширование пустой строки;
✅ Хеширование коротких и длинных строк;
✅ Коллизии — одинаковые строки дают одинаковый хеш;
✅ Лавинный эффект — изменение одного символа полностью меняет хеш;
✅ Анаграммы ("listen" vs "silent") — показывают, что порядок слов критичен.

Ключевые выводы:
1) Даже небольшое изменение входных данных приводит к кардинальному изменению хеша (лавинный эффект);
2) Одинаковые данные всегда дают одинаковый хеш (детерминированность);
3) Хеш имеет фиксированную длину, независимо от размера входных данных.

Часть 2: Простой блокчейн

Реализован блокчейн с Proof-of-Work (майнингом).

Демонстрируемые концепции:
✅ Неизменяемость — изменение данных в блоке ломает цепочку;
✅ Proof-of-Work — майнинг требует вычислительных ресурсов;
✅ Связность — каждый блок ссылается на предыдущий через previousHash;
✅ Валидация — автоматическая проверка всех хешей и ссылок.
